# PathDock

PathDock is a small browser extension for keeping a compact dock of frequently used sites.

This is a personal tool I built for my own workflow. It has no marketing plan, roadmap, or expected audience. I am sharing it as a developer manager who prefers to leave useful small tools in the open in case someone else wants to inspect, reuse, or adapt them.

## Features

- Save the current page to a popup dock.
- Open saved sites from the popup.
- Remove and reorder saved sites.
- Search YouTube from the popup.
- Search selected text with DuckDuckGo.
- Open selected text directly when it looks like a URL.
- Show and copy the current public IP address.
- Import and export saved sites with a local file.
- Works as a WebExtension for Edge and as a Safari Web Extension through the included Xcode project.

## Notes

The extension keeps the core data small: saved site URLs, titles, and their order. Browser sync is not required for import or export.
